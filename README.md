# Deep Learning Tutorial Project

## 项目简介

本项目面向深度学习初学者与入门实践者，围绕 **图像分类、图像生成、强化学习，以及基于 LLM 的 Agent** 四个方向，设计了一套循序渐进的教学与实验内容。

项目将从最基础的图像分类任务入手，帮助学习者掌握一个完整深度学习项目的核心流程，包括：

- 数据集准备与数据处理
- 模型搭建与训练
- 模型评估与结果分析
- 网络结构调整与实验对比

在完成图像分类基础后，项目会进一步扩展到图像生成任务，介绍多种经典生成模型的基本原理与实现方式，包括 **VAE、GAN、Diffusion**，以及 **基于回归的 LLM-Based 图像生成方法**。

除此之外，项目还将包含一个简单的强化学习示例，用于帮助学习者理解强化学习在控制任务中的基本思路。最后，项目将补充 **基于 LLM 的 Agent** 相关内容，帮助学习者理解大语言模型如何完成任务理解、规划、工具调用与执行。

---

## 项目目标

通过本项目，学习者将能够：

- 掌握图像分类任务的完整开发流程
- 理解深度学习中常见的数据处理与训练范式
- 学会对网络的尺寸、层数和基本结构进行调整
- 理解并实践常见图像生成模型
- 初步建立对强化学习基本思想的认识
- 理解基于 LLM 的 Agent 的基本概念与调用方式

## 数据集

本项目将使用 **The CIFAR-10 dataset** 作为图像分类与图像生成部分的基础实验数据集。
URL: https://www.cs.toronto.edu/~kriz/cifar.html

CIFAR-10 是一个经典的图像数据集，适合作为深度学习入门实验的数据基础，可用于：

- 图像分类基础训练
- 网络结构调整实验
- 小规模图像生成任务练习

> 强化学习与基于 LLM 的 Agent 部分将使用各自独立的环境或示例任务，不使用 CIFAR-10 数据集。

### 1. 数据集下载

首先创建用于存放数据集的目录，并进入该目录：

```bash
mkdir -p datasets
cd datasets
```

然后通过以下命令下载 CIFAR-10 数据集压缩包：

```bash
curl -O https://www.cs.toronto.edu/~kriz/cifar-10-python.tar.gz
```

除了命令行下载外，也可以前往官方网站手动下载该压缩包，并保存到当前目录。

下载完成后，执行以下命令解压数据集：

```bash
tar -xzf cifar-10-python.tar.gz
```

解压成功后，将得到 `cifar-10-batches-py` 文件夹，其中包含 CIFAR-10 数据集的 Python 版本文件，可用于后续实验。

### 2. 数据集查看

这是一个非常常用的数据集。为了便于使用，官方已经对数据集进行了图片整合和二进制格式处理。实际上，我们可以将这些整合后的二进制数据重新解析为图片进行查看。通常情况下，数据集中的样本大多以图片形式存在，因此这里主要是将二进制文件中的数据还原为图片，方便直观地观察样本内容。

这里我们从二进制文件中提取 **100 个样本** 进行预览。

项目中提供了两个脚本，分别用于不同方式的数据集查看：

- `datasets_preview.py`：包含完整处理细节的脚本，用于从二进制文件中逐步提取样本，并保存为图片。
- `datasets_preview_thirdparty.py`：基于第三方库（如 `torchvision`）实现的数据查看脚本，可直接加载并显示图片，使用更加简洁。

你可以根据需要选择合适的方式进行数据集预览。

## 项目内容

### 1. 图像分类

本部分将以图像分类任务为起点，帮助学习者建立对深度学习训练流程的整体认识，主要包括：

- 基础图像分类模型训练
- 数据处理模块化与库化
- 模型训练流程模块化与库化
- 网络尺寸调整
- 网络层数调整
- 网络结构的基础修改与实验

通过这一部分，学习者可以掌握从数据到训练、从模型搭建到评估分析的一整套基础流程。

---

### 2. 图像生成

本部分将系统介绍图像生成任务的基础概念，并围绕几类经典生成模型展开实现与分析，主要包括：

- 图像生成的基础概念
- VAE（变分自编码器）
- GAN（生成对抗网络）
- Diffusion（扩散模型）
- LLM-Based 自回归图像生成模型

其中，LLM-Based 图像生成模型将从“自回归生成”的角度进行介绍，即模型基于已有内容逐步预测下一个值，最终完成整张图像的生成。

通过这一部分，学习者将对现代生成式模型的核心思想建立基础认知。

---

### 3. 强化学习

本部分将介绍强化学习的基础概念，并结合简单示例帮助学习者理解强化学习在控制任务中的应用，主要包括：

- 强化学习的基础概念
- 强化学习中的状态、动作与奖励机制
- 一个简单的强化学习入门案例
- 基于案例实现基础的机器人控制任务

通过这一部分，学习者可以对强化学习的基本流程和核心思想形成直观理解。

---

### 4. 基于 LLM 的 Agent

本部分将介绍基于 LLM 的 Agent 的基本概念与调用方式，帮助学习者理解大语言模型如何以智能体的形式完成复杂任务，主要包括：

- 基于 LLM 的 Agent 基本概念
- 基于 LLM 的 Agent 调用方式
- Agent 中的任务拆解与流程规划
- Agent 的简单工具调用示例
- 利用基于 LLM 的 Agent 完成简单任务执行示例

通过这一部分，学习者将能够初步理解从“模型能力”到“智能体应用”的过渡方式。
